# Architecture Overview

## High-Level Design
- Frontend: React + Vite + Tailwind CSS for analytics dashboards and visualizations.
- Backend: Express REST API for data access, orchestration, and integration points.
- AI Services: FastAPI placeholder for future ML model serving.
- Data: `datasets/` for datathon-provided datasets and prepared artifacts.

## Initial Principles
- Modular services with clear API boundaries
- Environment-driven configuration
- Incremental, testable feature delivery
